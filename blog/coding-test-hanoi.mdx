## 풀이

하노이의 탑 규칙

1. 한 번에 하나의 원판만 옮길 수 있다.
2. 큰 원판이 작은 원판 위에 올 수 없다.

원리는 전체 원판을 다른 곳으로 옮기려면, 가장 밑에 있는 원판을 그곳으로 먼저 옮겨야 한다는 것.
따라서 가장 밑에 있는 원판을 제외한 탑을 목적지가 아닌 곳으로 옮긴다는 생각.

- 원판 n개를 c로 옮겨야 한다면
- n - 1개의 탑을 b로 옮기고 가장 밑에 있는 원판을 c로 옮긴다.
- b에 있는 탑을 c로 옮긴다.

```
function solution(number) {
    const answer = [];

    function hanoi(n, from, to, by) {
        if (n === 1)
            answer.push([from, to])
        else {
            hanoi(n - 1, from, by, to); // n - 1개의 탑을 by로 옮기고
            answer.push([from, to]) // n번째 칸을 to로 옮겨서 맨 밑에 깔고
            hanoi(n - 1, by, to, from); // by에 있는 탑을 to로 옮긴다
        }
    }

    hanoi(number, 1, 3, 2); // n개의 탑을 1에서 3으로 옮긴다.
    return answer;
}
```
